# Výpočet průměru známek

Jednoduchý JavaScript program, který ze seznamu známek vypočítá průměr a určí slovní hodnocení.

# Funkce
1. Vypočítá aritmetický průměr.
2. Rozhodne o výsledku (Vyznamenání / Prospěl / Neprospěl).

# Příklad
Vstup: `[1, 1, 2]`
Výstup: `Průměr: 1.33`, `Hodnocení: Vyznamenání`

# Spuštění
Otevři `index.html` v prohlížeči a podívej se do konzole (F12).
